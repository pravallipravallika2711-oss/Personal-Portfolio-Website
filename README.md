# Portfolio Website

This is a mobile-friendly personal portfolio built with HTML, CSS and Bootstrap.

How to view locally

1. Open `index.html` in a browser (double-click or use a local server).

Publish to GitHub Pages

1. Create a GitHub repository and push this folder (root should contain `index.html`).
2. In the repo settings, enable Pages from the `main` branch (or `gh-pages`) and choose the root folder.
3. Wait a minute вЂ” your site will be available at `https://<your-username>.github.io/<repo>`.
